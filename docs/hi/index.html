औद्योगिक यांत्रिक → औज़ार यांत्रिक → वेल्डर → मशीनिंग यांत्रिक → उत्पादन कर्मचारी → गुणवत्ता परीक्षक → मशीन ऑपरेटर → कसाई → खाद्य तकनीशियन → वस्त्र तकनीशियन → मोची → प्रिंट मीडिया डिज़ाइनर → पैकेजिंग मशीन मैकेनिक → असेंबली कर्मचारी → सूक्ष्म यांत्रिकी तकनीशियन → काँच फूँकने वाला → स्वर्णकार → असबाब कारीगर → संयंत्र मैकेनिक → रसायन संयंत्र ऑपरेटर →

ऊर्जा एवं आपूर्ति

विद्युत तकनीशियन ऊर्जा आपूर्ति → पावर प्लांट ऑपरेटर → नेटवर्क मोंटियर → विंड टरबाइन मेकाट्रॉनिक्स तकनीशियन → सोलर तकनीशियन → सप्लाई तकनीशियन → ऊर्जा सलाहकार → क्लाइमेट तकनीशियन → जिला हीटिंग तकनीशियन → बायोएनर्जी प्लांट ऑपरेटर → नेटवर्क योजनाकार ऊर्जा → नगर उपयोगिता कर्मचारी → अपशिष्ट निपटान विशेषज्ञ → अपशिष्ट जल संयंत्र कर्मचारी → मापन तकनीशियन ऊर्जा → ऊर्जा प्रबंधक → फाइबर-ऑप्टिक तकनीशियन → स्विचगियर निर्माता → जल प्रबंधक → विकिरण सुरक्षा अधिकारी →

सफाई एवं सुविधा प्रबंधन

भवन सफाईकर्मी → भवन देखभालकर्मी → सुविधा प्रबंधक → भवन तकनीशियन → कीट नियंत्रण विशेषज्ञ → औद्योगिक सफाईकर्मी → काँच एवं मुखौटा सफाईकर्मी → शीतकालीन सेवा कर्मचारी → स्वच्छता विशेषज्ञ → लिफ्ट तकनीशियन →

सफाई और सुविधा प्रबंधन

अलार्म सिस्टम इंस्टॉलर → हाउस मैनेजर → कंसीयर्ज → सफाई प्रबंधक → इंटीरियर फिटर → क्लीनरूम तकनीशियन → पार्किंग गैरेज कर्मचारी → एयर कंडीशनिंग मेंटेनेंस तकनीशियन → हरित क्षेत्र देखभालकर्ता → लॉकिंग/क्लोज़िंग सर्विस कर्मचारी →

व्यक्तिगत सेवाएँ

नाई → सौंदर्य विशेषज्ञ → मेकअप आर्टिस्ट → टैटू कलाकार → नेल डिज़ाइनर → मसाज थेरेपिस्ट → पर्सनल ट्रेनर → योग शिक्षक → स्टाइलिस्ट → अंत्येष्टि कर्मी → पशु-नाई → घर-गृहस्थी सहायक → बाल-देखभालकर्ता → शादी आयोजक → वैवाहिक सलाहकार → पैर-देखभालकर्ता → पोषण सलाहकार → जीवन सलाहकार → पालतू-जानवर देखभालकर्ता → मूविंग सहायक →

रियल एस्टेट और वास्तुकला

आर्किटेक्ट → इंटीरियर आर्किटेक्ट → रियल एस्टेट एजेंट → रियल एस्टेट मैनेजर → बिल्डिंग ड्राफ्ट्समैन → शहरी योजनाकार → लैंडस्केप आर्किटेक्ट → निर्माण विशेषज्ञ → रियल एस्टेट मूल्यांकनकर्ता → रियल एस्टेट क्लर्क →

रियल एस्टेट और आर्किटेक्चर

रियल एस्टेट प्रोजेक्ट डेवलपर → सर्वेक्षण तकनीशियन → स्मारक संरक्षण विशेषज्ञ → निर्माण वित्त सलाहकार → जनरल कॉन्ट्रैक्टर निर्माण → बिल्डर-डेवलपर → लाइट प्लानर → TGA प्लानर → अग्नि सुरक्षा योजनाकार → स्ट्रक्चरल इंजीनियर →
