Kami sangat serius dalam melindungi data pribadi dan profesional Anda. Penggunaan situs web kami dimungkinkan tanpa memberikan data pribadi (seperti nama atau alamat).

3. Hosting di GitHub Pages dan file log server

Situs web ini dihosting melalui layanan "GitHub Pages", yang dioperasikan oleh GitHub Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA. Penyedia secara otomatis mengumpulkan dan menyimpan informasi dalam apa yang disebut file log server, yang secara otomatis dikirimkan oleh peramban Anda. Ini biasanya meliputi:

  • Jenis peramban dan versi peramban
  • Sistem operasi yang digunakan
  • URL perujuk (halaman yang sebelumnya dikunjungi)
  • Nama host komputer yang mengakses (alamat IP yang dipersingkat/dianonimkan)
  • Waktu permintaan server

Data ini tidak dapat dikaitkan dengan orang tertentu dan tidak digabungkan dengan sumber data lain. Dasar pemrosesan data adalah Pasal 6 ayat 1 huruf f GDPR, yang mengizinkan pemrosesan data untuk menjaga kepentingan sah kami dalam penyediaan situs web yang aman dan bebas kesalahan secara teknis.

4. Cookie dan pelacakan

Kami tidak menggunakan cookie milik kami sendiri di situs web ini yang memerlukan persetujuan eksplisit. Kami tidak menggunakan alat pelacakan seperti Google Analytics untuk melindungi privasi Anda semaksimal mungkin.

5. Program mitra afiliasi

Kami menyematkan tautan ke buku-buku profesional dan kursus online terpilih di situs web kami. Dalam proses ini, kami berpartisipasi dalam program mitra. Jika Anda mengeklik tautan afiliasi tersebut dan melakukan pembelian di halaman tujuan, kami menerima komisi kecil. Bagi Anda, harga pembelian tidak berubah.

Untuk dapat menelusuri asal pesanan guna perhitungan komisi (disebut pelacakan konversi), mitra kami menggunakan cookie di peramban Anda atau memproses alamat IP Anda. Penyematan ini dilakukan berdasarkan kepentingan sah kami untuk membiayai layanan analisis gratis kami sesuai Pasal 6 ayat 1 huruf f GDPR.

Program Mitra Amazon

Kami adalah peserta dalam program mitra Amazon EU S.à r.l. Untuk menelusuri pesanan, Amazon menggunakan cookie yang memungkinkan Amazon mengenali bahwa Anda mengeklik tautan mitra di situs web kami. Program Afiliasi Udemy

Kami menautkan ke kursus pengembangan diri dari Udemy, Inc. Jika Anda mengikuti tautan ke Udemy, data pelacakan akan dikirim ke Udemy atau jaringan afiliasinya. Informasi lebih lanjut tentang perlindungan data di Udemy dapat Anda temukan dalam ketentuan privasi mereka.

6. Hak Anda sebagai subjek data

Dalam kerangka ketentuan hukum yang berlaku, Anda setiap saat berhak memperoleh informasi tanpa biaya tentang data pribadi Anda yang mungkin kami simpan, asal-usul dan penerimanya, serta tujuan pemrosesan data, dan juga berhak untuk memperbaiki atau menghapus data tersebut. Untuk itu, Anda dapat menghubungi kami kapan saja melalui alamat yang tercantum dalam pemberitahuan hukum.

7. Menghubungi kami

Jika Anda mengirimkan email kepada kami, pesan Anda beserta alamat pengirim hanya akan digunakan untuk memproses permintaan Anda. Tidak ada penerusan kepada pihak ketiga. Kami menegaskan bahwa transmisi data di internet (misalnya saat berkomunikasi melalui email) dapat memiliki celah keamanan.
