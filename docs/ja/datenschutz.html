当社は、お客様の個人的および職業上のデータの保護を非常に重視しています。お客様の個人データは、機密として、また適用されるデータ保護法および本プライバシーポリシーに従って取り扱います。当社ウェブサイトは、氏名や住所などの個人データを入力せずに利用できます。

C.O.R.E. Index テストに関する重要なお知らせ

心理測定評価の一環としてご回答いただくすべての回答、ならびに生成されたプロフィールおよびPDF資料は、お客様のWebブラウザ内でのみローカルに処理されます。これらの機微なテストデータが当社サーバーへ送信、保存、または分析されることは一切ありません。テスト進行状況の一時保存は、お客様のブラウザのローカルデータベース（localStorage）内でのみ行われ、テスト終了後に自動的に削除されます。

3. GitHub Pagesでのホスティングとサーバーログファイル

このウェブサイトは、GitHub Inc.（88 Colin P Kelly Jr St, San Francisco, CA 94107, USA）が運営する「GitHub Pages」サービスを通じてホスティングされています。提供事業者は、いわゆるサーバーログファイルに、ブラウザが自動的に送信する情報を自動的に収集・保存します。通常、これには以下が含まれます:

  • ブラウザの種類およびバージョン
  • 使用しているオペレーティングシステム
  • リファラーURL（直前に訪問したページ）

これらのデータは特定の個人に関連付けられるものではなく、他のデータソースと統合されることもありません。データ処理の法的根拠は、当社ウェブサイトを安全かつ技術的に問題なく提供するという当社の正当な利益を保護するためのデータ処理を認める、GDPR第6条第1項fです。

4. Cookieとトラッキング

当社ウェブサイトでは、明示的な同意を必要とする独自のCookieは使用していません。お客様のプライバシーを最大限に保護するため、Google Analyticsのようなトラッキングツールは使用していません。

5. アフィリエイト・パートナープログラム

当社ウェブサイトでは、厳選した専門書籍やオンラインコースへのリンクを掲載しています。これにより、当社はパートナープログラムに参加しています。アフィリエイトリンクをクリックして遷移先で購入された場合、当社は少額の報酬を受け取ります。お客様の購入価格は変わりません。

注文の出所を報酬計算のために追跡するため（いわゆるコンバージョントラッキング）、当社のパートナーはお客様のブラウザにCookieを設定するか、IPアドレスを処理します。この導入は、無料の分析サービスを維持するという当社の正当な利益に基づき、GDPR第6条第1項fに従って行われます。

Amazon パートナープログラム

当社は Amazon EU S.à r.l. のパートナープログラムに参加しています。注文の追跡のため、Amazon はCookieを使用し、当社ウェブサイト上のパートナーリンクをクリックしたことをAmazonが認識できるようにします。Amazonによるデータ利用の詳細は、Amazonのプライバシーポリシーをご覧ください。
