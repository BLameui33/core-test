C.O.R.E. Index は、あなたの仕事を妨げている無意識の葛藤を特定します。あなたの職種を選び、データ駆動型の8分間評価を開始してください。

私たちは色分けタイプもお世辞も提供しません。システム的な因果関係を分析します。

01. 生データ

評価は、現代心理測定学の科学的ゴールドスタンダードであるビッグファイブ・モデルに基づいています。

02. 影

私たちはあなたの強みだけでなく、あなたを妨害する存在不安や失敗への恐れを切り分けて測定します。

03. 結果

あなたは、どこで明確な境界線を引くべきか、あるいは交渉すべきかを客観的に示す、実行可能な行動ステップを受け取ります。
