01. 정밀한 데이터

평가는 현대 심리측정의 과학적 표준인 빅파이브 모델을 기반으로 합니다.

02. 그림자

우리는 당신의 강점만 측정하는 것이 아니라, 당신을 방해하는 존재 불안 또는 실패 불안을 분리해냅니다.

03. 결과
